## Quick Setup

```sh
# clone the project
git clone https://github.com/Circularorbit/electron-vite-vue-element-plus.git

# enter the project directory
cd electron-vite-vue-element-plus

# check out .npmrc file mirror source settings for you
cat .npmrc

# install dependency
npm install

# develop
npm run dev
```
